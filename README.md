# vs-preview

一個專為 Windows 系統設計的 VS Code 擴展，取代內建的多媒體資源預覽功能，提供**專業級的檢視體驗**與**便捷的操作工具**。目前首先支援圖片格式，未來將逐步擴展至更多類型的多媒體資源。

> **本擴展僅支援 Windows 系統。** 圖片剪貼簿等功能依賴 PowerShell 與 .NET Framework，其他作業系統無法使用完整功能。

---

## 核心功能

### 圖片檢視器

取代 VS Code 預設的圖片編輯器，提供專業級的圖片檢視與互動體驗。

| 功能名稱       | 功能描述                                                                               |
| -------------- | -------------------------------------------------------------------------------------- |
| 手勢操作支援   | 支援滑鼠/觸控板縮放、平移、慣性滾動，在右鍵選單中可快速重設視角                        |
| 吸管取色工具   | 內建 EyeDropper API 吸管工具，一鍵取色並自動複製色碼至剪貼簿                           |
| 圖片格式轉換   | 右鍵選單提供「導出為...」選項，可將圖片轉換並儲存為其他格式（PNG、JPG、WebP 等）       |
| 導出後快速檢視 | 導出完成後，通知中的「開啟檔案」按鈕可快速在編輯器中開啟新檔案，方便比較轉換前後的差異 |

### 圖片剪貼簿

提供進階的圖片複製功能，支援多種場景與格式。

| 功能名稱       | 功能描述                                                                            |
| -------------- | ----------------------------------------------------------------------------------- |
| 檢視器內複製   | 在圖片檢視器中使用 `Ctrl+C` 即可直接複製當前圖片                                   |
| 二進位資料複製 | 複製的是圖片二進位資料而非檔案路徑，可直接貼上至其他應用程式                        |
| 格式轉換支援   | 自動將 Windows 原生無法直接處理的格式（WebP、TIFF 等）轉換為 PNG 後複製，確保相容性 |
| 大型圖片進度   | 複製大型圖片時顯示即時進度條，提供清晰的視覺回饋                                    |

## 安裝指南

### 系統需求

- **作業系統**: Windows
- **VS Code**: 1.106.1 或更高版本

### 安裝步驟

1. 前往 [GitHub Releases](https://github.com/1ureka/vs-preview/releases) 下載最新的 `.vsix` 檔案

2. 在 VS Code 中安裝擴展：
   - 開啟命令面板（`Ctrl+Shift+P`）
   - 輸入並選擇 `Extensions: Install from VSIX...`
   - 選擇下載的 `.vsix` 檔案

3. 安裝完成後，開啟任何支援的圖片檔案即可自動啟用圖片檢視器

### 支援的圖片格式

`JPG` `JPEG` `PNG` `GIF` `BMP` `WebP` `TIFF` `TIF`

## 使用方法

### 檢視圖片

1. 在 VS Code 中直接開啟支援格式的圖片檔案，圖片檢視器會自動取代內建的預覽
2. 使用滑鼠滾輪或觸控板進行縮放，按住滑鼠左鍵進行平移
3. 右鍵開啟操作選單，提供以下功能：
   - **重設縮放與位置** — 一鍵恢復初始視角
   - **吸取顏色並複製** — 啟動吸管工具，點選螢幕上任意位置取色
   - **導出為...** — 選擇目標格式與儲存路徑，將圖片轉換導出

### 複製圖片

在檢視器中使用 `Ctrl+C`，圖片二進位資料將自動複製到系統剪貼簿，可直接貼上至 Photoshop、Word 等應用程式。

> [!TIP]
>
> WebP、TIFF 等 Windows 原生不支援的格式會自動轉換為 PNG 後複製，無需手動處理格式相容性問題。
